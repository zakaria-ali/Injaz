# ملف إنجاز المعلم — م. زكريا ولدعلي

موقع ويب لملف إنجاز المعلم في تخصص الحاسوب والشبكات — مدرسة محمود خليل أبو الرب المهنية.

## التشغيل

```bash
npm install
npm run dev
```

ثم افتح المتصفح على: http://localhost:5173

## البناء للإنتاج

```bash
npm run build
npm run preview
```

## الصفحات

- **الرئيسية**: الرؤية والرسالة والأهداف والسمات المهنية ومؤشرات الأداء
- **السيرة الذاتية**: وسائل التواصل ونبذة عني والخبرات والتعلم والمهارات والمشاريع (قابلة للتعبئة لاحقًا)
- **التخصص**: تعريف بتخصص صيانة أجهزة الحاسوب والشبكات
- **الصفوف**: الصف الأول ثانوي والصف الثاني ثانوي مع المواد الدراسية
- **الأعمال الكتابية**: فلتر حسب المادة والفصل — امتحانات يومية، شهرين، نهائي، خطط، تحضيرات
- **الوسائل التعليمية**: اختيار المادة والدرس لعرض عروض بوربوينت وفيديوهات ومواقع

## التعديلات اللاحقة

- **الصورة الشخصية**: ضع صورتك العرضية (أفقية) في مجلد `public` باسم `profile.jpg` — ستظهر في الصفحة الرئيسية (صورة مصغرة) وفي صفحة السيرة الذاتية (بحجم أكبر).
- **السيرة الذاتية PDF**: تم ربط زر التنزيل بملف `public/CV.pdf`. يمكنك استبدال الملف بتحديث محتوى السيرة لاحقًا.
- تحديث بيانات السيرة الذاتية (إيميل، هاتف، لينكد إن، جيت هب، ونبذة/خبرات/مهارات/مشاريع) من ملفك.
- وضع ملفات Word الفعلية في مجلد `public/files` وتحديث `src/data/writtenWork.js`
- وضع ملفات الوسائل التعليمية وتحديث `src/data/teachingAids.js`
